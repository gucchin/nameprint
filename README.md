# A4名札作成 Webアプリ (PWA)

A4サイズの用紙にぴったりと収まる縦書き名札を作成・印刷できるWebアプリケーションです。
iPhoneのホーム画面に追加してアプリとして使用することも可能です。

## 特徴
- **7枚配置**: A4縦向きに、縦長6枚＋横長（回転）1枚の計7枚を効率よく配置。
- **リアルタイム編集**: 氏名、学年・組を打ち込むと即座に全名札に反映。
- **カスタマイズ**: フォントサイズ（pt）の変更、フォントの種類（明朝・ゴシック・メイリオ）の選択が可能。
- **PWA対応**: iPhoneのSafariで「ホーム画面に追加」することで、オフラインでもネイティブアプリのように動作。
- **印刷最適化**: 印刷時は不要な操作パネルを自動で隠し、プレビュー通りの名札のみを出力。

## 使い方
1. `index.html` をブラウザで開きます。
2. 必要な情報を入力し、フォントサイズや種類を調整します。
3. 「印刷・PDF保存」ボタンを押して印刷します。
